# Blog Java – README Personal (Localhost Setup)

Este documento es **solo para uso personal**, para levantar el proyecto rápidamente en cualquier entorno local.

---

## 1. Requisitos en máquina

Debo tener instalado:

- Java 25 (Adoptium o similar)
- Maven 3.9+
- Docker Desktop
- IntelliJ IDEA (Community vale)

### Comprobación rápida

```bash
java -version
mvn -version
docker --version
